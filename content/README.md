# Content Files

This directory contains all editable content for your portfolio.

## Files

- **site-config.ts** - Personal info, hero text, about section, afterwork hobbies
- **projects.ts** - All case studies and project content

## How to Edit

1. Open the file you want to edit
2. Change the values (they're clearly labeled)
3. Save the file
4. Your changes appear automatically in dev mode (`npm run dev`)

For detailed instructions, see [CONTENT-EDITING.md](../CONTENT-EDITING.md) in the root directory.
